### 1. Installation
```bash
git clone https://github.com/moinecalixte-cyber/aequitas.git
cd aequitas
cargo build --release
```

### 2. Run a Full Node
Join the decentralized mesh.
```bash
./target/release/aequitas-node run
```

Put your GPU to work and secure the network.
```bash
# Add your address to miner.toml
```
